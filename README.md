# Hello Coffee - Point of Sale (POS) System ☕

Sistem kasir (Point of Sale) sederhana berbasis web yang dirancang untuk kedai kopi. Proyek ini dibangun menggunakan HTML, CSS, dan Vanilla JavaScript.

## Fitur Utama ✨
- **Sistem Login Kasir:** Otentikasi sederhana untuk membedakan kasir dan admin.
- **Manajemen Pesanan (Dine In & Take Away):** Pencatatan pesanan lengkap dengan nama pelanggan dan nomor meja.
- **Keranjang Belanja (Cart):** Perhitungan otomatis subtotal, pajak (10%), diskon, dan total pembayaran.
- **Kategori & Pencarian Menu:** Navigasi menu yang mudah menggunakan kategori (Kopi, Non-Kopi, Makanan, Snack) dan fitur pencarian.
- **Sistem Pembayaran:** Dukungan berbagai metode pembayaran (Tunai, QRIS, Transfer, Kartu) beserta kalkulator kembalian otomatis.
- **Cetak Struk:** Simulasi pencetakan struk transaksi untuk pelanggan.
- **Admin Panel:**
  - Dashboard statistik penjualan.
  - Manajemen Menu (Tambah, Edit, Hapus produk).
  - Riwayat Transaksi.
  - Laporan penjualan.
- **Desain Responsif:** Tampilan UI sudah disesuaikan agar berjalan lancar di perangkat *Desktop*, *Tablet*, dan *Mobile (Handphone)*.

## Teknologi yang Digunakan 💻
- HTML5
- CSS3 (Vanilla CSS dengan CSS Variables)
- Vanilla JavaScript (Tanpa framework tambahan)

## Cara Menggunakan 🚀
1. *Clone* atau unduh *repository* ini:
   ```bash
   git clone https://github.com/SyafiqSiregar/Hello-Coffee.git
   ```
2. Buka folder proyek.
3. Jalankan file `index.html` langsung di *browser* Anda (Google Chrome, Firefox, Safari, dll).
